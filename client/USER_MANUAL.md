# LUNORA Customer User Manual

This manual guides customers on shopping, registering, checking out, and tracking orders.

## 🛒 1. Placing a Cash on Delivery (COD) Order
1. Select items, choose color and quantity, and click **Add to Cart**.
2. Click the cart icon in the top right to verify order summary.
3. Click **Proceed to Checkout**.
4. Input shipping address and contact phone number.
5. Select **Cash on Delivery (COD)** as the payment method.
6. Input coupon code `WELCOME200` to apply flat ₹200 off your order.
7. Click **Place Order**. You will receive an email confirmation with an attached PDF invoice.

## 💳 2. Placing a Card / UPI Order (Razorpay)
1. Complete steps 1-4 above.
2. Select **UPI / Card** as the payment method.
3. Tap **Pay securely**. The Razorpay checkout modal opens.
4. Input test credentials in Sandbox mode or live payment details.
5. Upon signature authorization, you will be redirected to the `/order-success` confirmation dashboard.

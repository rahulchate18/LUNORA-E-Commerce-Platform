# LUNORA Support Guide

Welcome to the LUNORA support guide. If you have questions, run into issues during deployment, or need custom additions:

## 📞 Support Channels

### 1. Developer Documentation
* Review the technical documentation inside the [`docs/`](docs/) folder.
* Consult [`DEPLOYMENT.md`](DEPLOYMENT.md) for Render and Vercel hosting.
* Review variable details in [`ENVIRONMENT.md`](ENVIRONMENT.md).

### 2. GitHub Issue Tracker
* Search existing issues to see if your query has already been resolved.
* Open an issue with clear replication details, screenshots, and server logs.

### 3. Dedicated Concierge Desk
* Paying enterprise clients can contact the dedicated LUNORA concierge desk at `support@lunora.com`.
* Response time is guaranteed within 12 hours for critical severity production bugs.

# LUNORA Client Onboarding Checklist

Use this checklist to coordinate domain and cloud transfers:

## 1. Domain & DNS Control
- [ ] Transfer DNS access to the client.
- [ ] Configure MX/SPF domains records for Resend/SMTP email validation.
- [ ] Bind custom domain names to the storefront Vercel deployment project settings.

## 2. Infrastructure Accounts Transfer
- [ ] Transfer ownership of the **MongoDB Atlas** database cluster project settings.
- [ ] Configure client billing details on **Cloudinary**.
- [ ] Connect client bank routing details to **Razorpay Live Mode** account dashboard.
- [ ] Set up monthly maintenance retainers.
